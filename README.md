# 🏀 Perfect Directions

**How Sean Saved Steph's Night in Las Vegas**

An illustrated kids' storybook web app about a 10-year-old Chinese American boy named Sean who meets a lost basketball star on the Las Vegas Strip and gives him perfect directions back to his hotel — just in time for the next night's game.

## Features

- 📖 8-page illustrated storybook
- 🔊 Audio narration (male voice, Edge TTS)
- 🏀 Golden State Warriors themed UI
- 🌙 Vegas night scenes + SF arena illustrations
- ⌨️ Keyboard navigation (← → Space)
- 📱 Responsive design
- ✨ Smooth animations & transitions

## Tech Stack

- React 18 + Vite
- Tailwind CSS 3
- Edge TTS (audio narration)
- Cloudflare Workers AI (FLUX image generation)

## Development

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
```
